BUTTON

Для использования кнопки в своём проекте вам необходимо на своей странице разместить тег <button> со следующими атрибутами:

  • handler - в данном атрибуте указывается путь, по которому будет отправлен запрос на сервер об выполнении какой-либо функции. Например: <button handler="shot"></button>
  • onResponse* - в данном атрибуте указывается куда придёт ответ от сервера об выполнении какой-либо функции. Например: <button handler="shot" onResponse="$('#resp-btn').text(x);"></button>
  • clickable - данный атрибут указывает на то, что запрос на сервер будет выполняться по клику на кнопку.

    • *необязательный атрибут

SLIDER

Для использования ползунка в своём проекте вам необходимо на своей странице разместить тег <input type="range"/> со следующими атрибутами:

  • min - в данном атрибуте указывается минимальное значение ползунка.
  • max - в данном атрибуте указывается максимальное значение ползунка.
  • value* - в данном атрибуте указывается текущее значение ползунка.
  • step - в данном атрибуте указывается с каким шагом будет изменяться значение ползунка.
  • handler - в данном атрибуте указывается путь, по которому будет отправлен запрос на сервер об выполнении какой-либо функции. Например: <input type="range" min="12" max="18" step="1" value="" handler="temperature"/>
  • onResponse* - в данном атрибуте указывается куда придёт ответ от сервера об выполнении какой-либо функции. Например: <input type="range" min="12" max="18" step="1" value="" handler="temperature" onResponse="$('#resp-slider').text(x);"/>
  • changeable - данный атрибут указывает на то, что запрос на сервер будет выполняться по изменению его значения.

*необязательный атрибут

SELECT

Для использования селекта в своём проекте вам необходимо на своей странице разместить тег <select>, внутри которого должны быть теги <option>, со следующими атрибутами:

  • handler - в данном атрибуте указывается путь, по которому будет отправлен запрос на сервер об выполнении какой-либо функции. Например: <select handler = "plant">
    <option>Не выбрано</option>
    <option>Цветы</option>
    <option>Овощи</option>
    <option>Фрукты</option>
    </select>
  • onResponse* - в данном атрибуте указывается куда придёт ответ от сервера об выполнении какой-либо функции. Например: <select handler = "plant" onResponse="$('#resp-select').text(x);">
    <option>Не выбрано</option>
    <option>Цветы</option>
    <option>Овощи</option>
    <option>Фрукты</option>
    </select>
  • changeable - данный атрибут указывает на то, что запрос на сервер будет выполняться по изменению его значения.

*необязательный атрибут
